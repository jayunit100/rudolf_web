The Pegasus : Word Enrichment Calculator 

	  Pegasus-WE is a simple web service which takes, as input, a url, and outputs a json map of words to enrichments.  
	  It is written in clojure. It can be called as a service : simply add the url whose word enrichments you want to calculate, 
	  and add it as the first parameter to the request.  Some examples are below :  
		http://www.rudolflabs.com/tools/wordenrichment_url?url=http://zimmer.csufresno.edu/~fringwal/myths2.txt
		http://www.rudolflabs.com/tools/wordenrichment_url?url=http://nhregister.com/articles/2012/06/06/news/doc4fcfcf1aae7b2606017569.txt
		http://www.rudolflabs.com/tools/wordenrichment_url?url=http://projects.chass.utoronto.ca/emls/iemls/shaksper/logs/98log.txt
